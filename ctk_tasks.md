## Задание 3. Надёжен ли пароль?

Создайте приложение, которое показывает, надёжен ли ваш пароль. Пароль считается надёжным, если его длина составляет не менее 5 символов, и если он содержит хотя бы одну заглавную букву и хотя бы одну цифру. Создайте два поля для текста CTkEntry(): одно - для ввода пароля пользователем, второе - для вывода результата проверки (надёжный/слабый).

## Задание 5. Конвертер величин

Создайте приложение для перевода из одной величины измерения в другую. Для этого создайте два выпадающих списка: "Из чего?" и "Во что?". Каждый список должен содержать следующие значения - метры, футы, килограммы, фунты. Также добавьте текстовое поле для вывода результата и кнопку "Перевести" для получения результата.

Примечание: 1 кг = 2.205 фунта, 1 м = 3.28 фута.
