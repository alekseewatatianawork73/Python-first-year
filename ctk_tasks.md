# Задания по библиотеке Customtkinter (начальный уровень)

## Задание 1. Подсчёт площади прямоугольника

Напишите программу, которая будет подсчитывать площать прямоугольника по его сторонам. Создайте три поля для текста CTkEntry(): два поля для ввода сторон прямоугольника и одно - для вывода его площади. Площадь будет подсчитываться при нажатии на кнопку "Найти площадь".

## Задание 2. Конвертер валют

Создайте приложение для конвертации рублей в доллары и евро. Создайте поле для ввода суммы в рублях и две кнопки: "Конвертировать в USD" и "Конвертировать в EUR" (1 долл. = 77,19 руб., 1 евро = 91,56 руб.). Можете добавить по желанию текстовые метки, изенить цвета и оформление.
