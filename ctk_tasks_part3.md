## Задание 1. Обмен информацией

Напишите приложение для обмена информацией между вкладками. Для этого создайте окно с двумя вкладками: "Отправитель" и "Получатель".

- Вкладка "Отправитель" должна содержать текстовое поле для ввода сообщения и кнопку "Отправить".
- Вкладка "Получатель" должна содержать текстовое поле для вывода отправленного сообщения (текст в поле нельзя изменять) и кнопки "Показать сообщение" и "Очистить поле".

Логика приложения:
- При нажатии на кнопку "Отправить" введённое сообщение из вкладки "Отправитель" сохраяется в программе, а во вкладке "Получатель" появляется надпись в текстовом поле "У Вас новое сообщение!".
- При нажатии на кнопку "Показать сообщение" во вкладке "Получатель" в текстовом поле появляется отправленное сообщение (используйте для хранения сообщения глобальную переменную).
- При нажатии на кнопку "Очистить поле" весь текст из текстового поля во вкладке "Получатель" удаляется.

## Задание 2. Работа с текстом

Создайте приложение для анализа текста, которое работает с текстом, введённым пользователем, и предоставляет небольшую статистику по нему. Создайте три вкладки:

- Вкладка "Ввод текста" должна содержать текстовое поле для ввода вашего текста (Textbox) и кнопку "Сохранить".
- Вкладка "Анализ текста" должна содержать надписи о количестве символов и слов в тексте, а также кнопку "Рассчитать".
- Вкладка "Дополнительно" содержит текстовое поле (Entry) для ввода слова, кнопку "Найти" и надпись с результатом.

Логика приложения:
- При нажатии на кнопку "Сохранить" введённый текст сохраняется в программе (можно использовать глобальную переменную для хранения).
- При нажатии на кнопку "Рассчитать" во вкладке "Анализ текста" подсчитывается количество символов в вашем тексте с пробелами и без пробелов, а также количество слов.
- При нажатии на кнопку "Найти" во вкладке "Дополнительно" нужно посчитать, сколько раз введённое слово встречается в тексте и отобразить результат.

Подсказка. Для подсчёта длины текста, количества слов, поиска слов необходимо вспомнить тему "Строки" из базового курса (методы строк): https://docs.google.com/presentation/d/1yTY0-kw71m3DiWgX0XAzj4Fx1kHdgjB5sfd9-BTkQsY/edit?usp=sharing

## Задание 3. Домик на холсте

Создайте приложение, которое изображает домик на холсте двумя способами: с помощью геометрических фигур (прямоугольники, треугольники, линии) и используя готовую картинку. 

В вашем окне должны находиться следующие виджеты:

- холст Canvas для отображения домика
- выпадающий список (Combobox) для выбора вида домика: рисунок или картинка
- кнопка "Показать домик"
- кнопка "Скрыть домик"

Логика работы приложения:

- При нажатии на кнопку "Показать домик" на холсте появляется изображение домика. В зависимости от выбора в выпадающем списке это будет уже готовая картинка из интернета или нарисованная при помощи геометрических фигур.
- При нажатии на кнопку "Скрыть домик" холст становится пустым.

Ссылка на картинку домика: https://drive.google.com/file/d/1iwHUmbT6gg0uN74NGVV8B8GMRbGbiNKl/view?usp=sharing

Подсказка. Для получения значения из выпадающего списка используйте метод _get()_. Для рисования домика, как вариант, можно использовать методы _create_rectangle()_, _create_polygon()_ или любые другие на ваш выбор. Для удаления элементов с холста используется метод _delete()_.
