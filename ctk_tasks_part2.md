## Задание 1. Анкета с фреймами

Создайте главное окно с заголовком «Регистрация пользователя». Разместите в нём три фрейма:

Фрейм «Личные данные:
- Текстовое поле для ввода имени
- Текстовое поле для ввода возраста
- Выпадающий список с выбором пола: «Мужской», «Женский»

Фрейм «Интересы» (светло-зелёный фон, граница толщиной 2):
- Три чекбокса (CTkCheckBox): «Спорт», «Творчество», «Программирование»

Фрейм «Действия» (серый фон, без границы):
- Кнопка «Показать анкету»
- Кнопка «Очистить»

Все элементы должны быть расположены при помощи grid().

Дополнительно:
- При нажатии на «Показать анкету» выводите собранные данные в отдельную надпись под фреймами.
- Кнопка «Очистить» должна очищать все поля и сбрасывать чекбоксы.
