## Задание 1. Анкета с фреймами

Создайте главное окно с заголовком «Регистрация пользователя». Разместите в нём три фрейма:

Фрейм «Личные данные:
- Текстовое поле для ввода имени

## Задание 2. Окно настроек

Создайте главное окно надписью «Текущие настройки: по умолчанию» и кнопкой «Открыть настройки». При нажатии на кнопку должно открываться новое окно (нельзя открыть два таких окна одновременно!). В окне настроек разместите следующие элементы:

- Переключатели темы (CTkRadioButton): «Светлая» и «Тёмная»
- Один чекбокс «Показывать подсказки»
- Выпадающий список с размером шрифта: «Маленький», «Средний», «Большой»

При нажатии «Сохранить» окно настроек закрывается, а на главном окне обновляется надпись с текущими настройками (например, «Тема: dark, подсказки: открыты, шрифт: средний»). Если нажать «Отмена» или закрыть окно крестиком — изменения не применяются.
Если открыть окно настроек снова, в нём должны отображаться уже сохранённые настройки.

## Задание 3. Два фрейма - один выбор

Напишите приложение для рисования различных геометрических фигур. Для этого создайте главное окно и разместите в нём два фрейма. 

Левый фрейм - выбор фигуры и цвета фона:

- Выпадающий список с тремя элементами (цветами фона): «Белый», «Зелёный», «Желтый»
- Радиокнопки для выбора фигуры: «Круг», «Квадрат», «Звезда»

Правый фрейм - отображение результата:

- Фрейм меняет цвет фона в зависимости от выбора в выпадающем списке
- Внутри прямоугольника надпись, которая меняет форму в зависимости от выбора радиокнопки ("🔴", "🟦", "⭐")

Итак, при нажатии на кнопку цвета правый фрейм меняет цвет фона (параметр command в выпадающем списке). При выборе радиокнопки меняется текст надписи внутри (параметр command для радиокнопок).

Примечание. Не забудьте, что радиокнопки нужно обязательно объединять в группу, т. е. создавать переменную для группы и указывать ёё в свойстве _variable_ для каждой радиокнопки. Также помните, что у функции выпадающего списка, в отличие от кнопки, есть начальные данные в скобках - это выбор, который мы сделали. Если что-то забыли, просмотрите теорию по темам "Чекбоксы и радиокнопки" и "Выпадающие списки".

## Задание 5. Цветовой миксер

Напишите приложение для подбора цвета фона в формате RGB:
- создайте три ползунка внутри фрейма для управления цветом фона: Красный (R), Зелёный (G), Синий (B)
- значения каждого ползунка от 0 до 255 (как в RGB)
- создайте надпись для отображения результата, например, "Выбранный цвет в формате RGB: (0, 255, 255)"
- создайте фрейм с подобранным цветом фона

При изменении значений ползунков надпись и цвет фона фрейма для результата должны меняться.

**Как задать цвет фрейма в формате RGB?** У вас будут значения ползунков для красного, зеленого и синего цветов: r, g, b. Поменять цвет фрейма frm на выбранный цвет rgb можно при помощи следующей строки:

```
frm.configure(fg_color=f'#{r:02x}{g:02x}{b:02x}')
```

## Задание 7. Калькулятор чаевых

Напишите приложение для расчёта суммы чаевых в чеке. Для этого создайте окно с:

- полем ввода «Сумма счёта (в рублях):»;
- ползунком «Чаевые: 0%» со значениями от 0% до 50%;
- кнопкой «Рассчитать чаевые»;

И фрейм в этом окне для отображения результата с:

- надписью «Чаевые: 0 руб.» - будет менять значение при нажатии на кнопку;
- надписью «Итого: 0 руб.» - будет менять значение при нажатии на кнопку.

При нажатии на кнопку, в зависимости от введённой суммы и значения ползунка, пересчитываются чаевые и итоговая сумма. Пример:

- Сумма счёта: 1000 рублей, Чаевые: 20% => Результат: _чаевые = 1000 * 0.2 = 200 рублей, итоговая сумма = 1000 + 200 = 1200 рублей._

## Задание 8. Музыкальный плеер с регулировкой громкости

Создайте приложение для управления фоновой музыкой, которое позволяет включать/выключать музыку и регулировать громкость.

В вашем окне должны располагаться следующие элементы:

- Фрейм управления музыкой. Внутри фрейма должны быть размещены:
  - Надпись с текущим статусом музыки («Музыка играет» или «Музыка выключена»)
  - Надпись с текущей громкостью (например, «Громкость: 50%»)
- Кнопка для включения/выключения музыки. При нажатии:
  - Если музыка играет → выключаем её, текст кнопки меняется на «Включить музыку», надпись во фрейме меняется на «Музыка выключена»
  - Если музыка выключена → включаем её, текст кнопки меняется на «Выключить музыку», надпись во фрейме меняется на «Музыка играет»
- Ползунок для регулировки громкости (значения от 0% до 100%)

Вы можете скачать свой музыкальный файл или взять готовый по ссылке: https://drive.google.com/file/d/1avvt8UH8HMB8g5TNeaBwVfoYZUQAJUYy/view?usp=sharing 

**Подсказка:** чтобы не прописывать путь к файлу при загрузке, а только его название, сохраните файл рядом с вашей программой в той же папке. Для проверки, играет ли музыка, воспользуйтесь методом **mixer.music.get_busy()**. Для установки громкости звука подойдёт метод **mixer.music.set_volume()**.

## Задание 9. Таймер с сигналом

Создайте приложение-таймер для обратного отсчёта времени с сигналом в конце.

В вашем окне должны располагаться следующие элементы:

- Фрейм с отсчётом времени от 15 до 0 секунд
- Кнопка для запуска таймера

Как работает таймер?
- При нажатии на кнопку начинается отсчёт секунд от 15 до 0
- Каждую секунду надпись обновляется (используйте метод after)
- Когда счёт доходит до 0, четыре раза воспроизводится звуковой сигнал
